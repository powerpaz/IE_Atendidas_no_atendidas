README (Datos para coroplético + puntos IE)

A) Coroplético cantonal
- Archivo: data/NBI_Canton.csv
- Columnas mínimas:
  - DPA_CANTON (ejemplo: 0804)
  - NBI (ejemplo: 0.615)

B) Puntos IE
- Archivo: data/IE_Datos.csv
- Columnas mínimas:
  - lat, lon
  - AMIE
  - NOM_INSTITUCION_EDUCATIVA
  - DPA_CANTON
  - NOM_SOSTENIMIENTO
  - Mantenimiento C_I
  - Total Casos
  - Total estudiantes otras nacionalidades

Notas
- Puedes usar ';' o ',' como separador. Si tu Excel usa ';', guarda como CSV y listo.
- El campo NBI también se acepta como "% NBI_Cantón" o "% NBI_Cantón**".
